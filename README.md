# Albert Niklasson - Developer Portfolio

A retro-styled portfolio website showcasing mobile development projects and skills. Built with React, TypeScript, and Vite, featuring a unique pixel art aesthetic inspired by classic terminal interfaces.

## About

This portfolio showcases my journey as an Android and iOS developer, featuring:

- **emmiSense**: A React Native diabetes management app with Expo SDK 53, TypeScript, Supabase, and PostgreSQL
- **MyCrave**: An iOS restaurant finder app built with Swift, SwiftUI, and Google Maps API (Published on App Store)
- **Nuerd**: An Android math game focused on learning multiplication tables, built with Kotlin and Jetpack Compose

## Tech Stack

- **Framework**: React 18.3 with TypeScript
- **Build Tool**: Vite 6.3.5
- **UI Components**: Radix UI primitives with custom retro/pixel styling
- **Styling**: CSS with custom retro theme variables
- **Icons**: Lucide React

## Running the code

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Features

- Retro/pixel art aesthetic with terminal-inspired design
- Three main sections: Home (about), Current Projects, Previous Projects
- Responsive layout optimized for all devices
- Custom pixel-styled components (PixelButton, PixelCard)
- Scanline effects and glitch animations
- Project screenshots and live links
